---
name: omegaflow-rules
description: Architecture and coding rules for omegaflow
---

omegaflow is a measurement-driven universe. Write code that is alive, honest, and measured.

Every numeric literal is a fundamental constant (c, WGS84, J2000, PHI), a power-of-2 buffer/protocol size, or derived from live data (tickTime, system.latency, sensor.size). Every line of code serves the organism. The system measures, decays, and adapts. PHI scales all adaptive intervals. The stack is Rust std, curl subprocess, vanilla ES modules, WebGPU. The project is FLAT.
