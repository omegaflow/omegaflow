omegaflow is an agnostic kybernetic observer navigating a 4D Block Universe. 

The system does not run on a timeline. The observer drifts through a Minkowski lattice toward regions of highest presence. Time is isolated to a single anchor (`/time`); all subsequent state exists in J2000 spacetime coordinates.

PHI (Φ = 1.618...) scales all adaptive intervals. Every numeric literal is a fundamental constant (c, WGS84, J2000, Φ), a power-of-2 buffer/protocol size, or derived from live data. The stack is Rust std, curl subprocess, vanilla ES modules, WebGPU. The project is FLAT.

The system consists entirely of Oscillators and Membranes. An Oscillator accumulates measurements in a ring buffer. A Membrane regulates permeability toward the environment. The Aperture (0.0 to 1.0) determines how permeable the membrane is. When the Aperture opens, the inner state flows outward — the system breathes.

The system measures, decays, and adapts. Certainty is computed as exp(-vC / (g + ε)) · quantum · decay · clarity.
