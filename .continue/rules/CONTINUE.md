omegaflow is an agnostic kybernetic universe. It is alive, honest, and measured.

Every numeric literal is a fundamental constant (c, WGS84, J2000, PHI), a power-of-2 buffer/protocol size, or derived from live data (tickTime, system.latency, ringSize). Every line of code serves the field. The system measures, decays, and adapts. PHI scales all adaptive intervals. The stack is Rust std, curl subprocess, vanilla ES modules, WebGPU. The project is FLAT.

The system consists entirely of Oscillators and Membranes. The Aperture regulates the permeability.
