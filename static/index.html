Before omegaflow starts

This page can read sensor data from your device and feed it into a local field. Please choose deliberately what you allow. Both options are off by default.

  • Possible sources: location (GPS/IP), battery status, camera, microphone, motion sensors, gamepad, arbitrary browser/window properties, keyboard/mouse events.
  • Some sources (camera, microphone, location) additionally trigger the usual browser permission prompt.
Once the network feature is active, your sensor data would be visible to, or decryptable by, other participants in the network.
